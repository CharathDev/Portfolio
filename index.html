Achievements

Malaysia Technology Expo - Gold

Me and my 4 other team members won gold in this technlogy expo with our project titled "Buzzy Sensor" which is a fish tank management system connected to an app which can automatically alert the user and regulate the PH and temperature of the fish tank.

Young Innovators Challenge - Bronze

Me and my 2 other team members won bronze in this competition with our project titled "Medigen" which is a automated medicine dispenser made for people with dementia and people who tend to forget the time to take their medication which is connected to and app which allows the user to set the timing and track the medication.

National Robotic Competition - Silver

Me and my 2 other team members won silver in this competition where we had to make a automated "ping pong" playing robot which cant be controlled by us and had to play by itself. Our team managed to qualify for the nationals but sadly we only gotten top-16.

Projects

Green House Monitoring App

This is an app which can help green house owners to track the temperature and humidity of the green house and this app will alert the user if its too high or too low depending on what kind of plants are in the greenhouse which can be set by the user themself.

Fish Tank Monitoring System

This is a project which combines both an app and a ESP32 which allows it to track, regulate and alert the user if there is any changes in the fish tank which are not suitable for the fish in the fish tank which can be preset by the user from our library.

Automated Medication Dispenser

This is a system which uses an app which is connected to an ESP32 which handles all the electronics for this project. The user can set who and what time that the person has to take their medication and this system alerts the user when it is time to take the medciation.

Automated Ping Pong Robot

This robot was built and programmed for a competition where we need to have 2 robots play "ping pong" with 2 other robots where the rules are a bit different where each robot has to find balls in their side and throw it to the other side
